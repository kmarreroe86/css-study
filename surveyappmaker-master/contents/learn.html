How does SurveyAppMaker work?

Query Types Elements

Header / Title

Allow users to insert text as header or paragraph information.

Text Line

Allows users to enter one line word answers using screen keypad. Minimizing the use of this query type minimizes errors!

Text Box

Allows users to enter multiple lines word answers using screen keypad.

Decimal Number

Use this field when the answer will be a decimal number. For example, “What is the temperature in ℉?”

Integer numbers

Use this field when the answer will be a whole number. For example, “How many mosquitoes did you find in the trap?”

Dropdown list

Allows users to select from a list of answers provided. It can be of Single or Multiple selection.

Checkbox

Allows users to select from answers provided in check boxes items format. It can be Single or Multiple selection.

Radio button

Allows users to select only one option from a list of answers provided.

Barcode Scan

Scan barcode using your device. Eliminate manual data entry mistakes and standardize data collection and faster.

QR code Scan

Scan QR code using your device. Eliminate manual data entry mistakes and standardize data collection and faster.

Photo Capture

Take a picture to get evidences.

Location place

Allows users to select administrative divisions inside the country. Eliminating data entry mistakes.

Date-Time

Allow users to select date from electronic calendar, including current time.

Questions Block

Allow users to introduce selected questions to ask different times dynamically. Example: Mosquito traps collection to identify species found and count it by sex. You can find different species on different traps.

Hours

Facilitate to enter hours in appropriate format.

Signature

Allow users to include capture signatures wit device. You don’t need another special device
